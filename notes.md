## HTML

- HTML -> HyperText Markup language

- **HyperText** -> Enriched text, that has a structure, not your regular boring text
- **Markup** -> means of annotating documents / text by specifying and their format
- **language** -> *(programming language)*

- HTML pages -> Usually run on the browser natively

## Anatomy of a HTML Webpage

- "<html>" , "</html>" tag -> This indicates to the browser that your file is to be rendered as a web document.
- "<!DOCTYPE html>" -> This indicates to the browser to utilize latest HTML features.
- '<head> , </head> ->  Metadata about the web page.
- '<body>' , </body> -> Contains all the components of your web page.
